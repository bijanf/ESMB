# Code of Conduct

## Our pledge

We want participation in the Chronos-ESM community to be a welcoming, respectful,
and harassment-free experience for everyone, regardless of background or identity.

## Our standard

This project adopts the **Contributor Covenant**, version 2.1, as its code of conduct.
Please read the full text, which describes expected behavior, unacceptable behavior,
scope, and enforcement guidelines:

- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain-text/Markdown source: https://github.com/EthicalSource/contributor_covenant

In short: be kind and constructive, assume good faith, welcome newcomers, and accept
feedback gracefully. Behavior that demeans, harasses, or excludes others is not
acceptable in any project space (issues, pull requests, discussions, and any
related communication).

## Scope

This Code of Conduct applies within all project spaces and also when an individual is
officially representing the project in public spaces.

## Reporting and enforcement

If you experience or witness unacceptable behavior, please report it confidentially to
the project maintainer:

- **Bijan Fallah** — bijan.fallah@gmail.com

Reports will be reviewed and investigated promptly and fairly. The maintainer is
obligated to respect the privacy and security of anyone who reports an incident.
Enforcement follows the Contributor Covenant's Enforcement Guidelines (correction →
warning → temporary ban → permanent ban), applied proportionately.
